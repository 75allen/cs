	clock() / CLOCKS_PER_SEC is a time in seconds.
clock()是基于时钟计时单元(clock tick)的这个东西来计数的。一个clock tick不是CPU的一个时钟周期，而是C/C++的一个基本计时单位，因此只与编译器有关。在TC2.0中硬件每18.2个滴答是一秒，在VC中硬件每1000个滴答是一秒，在标准POSIX中定义为1000000个滴答为一秒。
